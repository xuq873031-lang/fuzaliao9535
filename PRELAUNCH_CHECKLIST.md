# 上线前关键回归检查清单（P0）

- [ ] `APP_ENV=production`
- [ ] `DATABASE_URL` 指向 PostgreSQL（非 SQLite）
- [ ] `ENFORCE_NON_SQLITE_IN_PRODUCTION=true`
